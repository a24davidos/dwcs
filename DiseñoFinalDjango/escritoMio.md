## **Note Management Web App – Diseño**

Para este proyecto he decidido utilizar mi proyecto anterior de PHP y adaptarlo a este nuevo Framework.

Mi idea para este proyecto es crear una aplicación web que permita crear y gestionar notas personales. Los usuarios pueden registrarse, iniciar sesión, crear, editar, eliminar y organizar sus notas.

Cada nota puede pertenecer a una o mas categorías que el usuario puede crear, y también puede subir una imagen e ncada nota.
